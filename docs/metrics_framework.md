# Metrics Framework

## On-field metrics
- Wins
- Losses
- Run differential
- Pythagorean wins
- Strikeouts allowed
- Attendance

## Payroll / contract metrics
- Total payroll
- Maximum salary
- Median salary
- Top-1 salary share
- Top-3 salary share
- Top-5 salary share
- Salary Gini coefficient

## Efficiency metrics
- Payroll per win
- Wins per $10M
- Run differential per $10M
- Pythagorean gap = actual wins - expected wins

## High-priority next metrics

### Player value
- WAR
- Cost per WAR
- WAR concentration
- % team WAR from pre-arbitration players

### Roster construction
- Average hitter age / pitcher age
- % payroll on top decile contracts
- % payroll on injured or low-value players
- Dead money index

### Team quality beyond record
- BaseRuns wins
- One-run game overperformance
- Bullpen leverage efficiency
- Defensive run prevention indicators

### Comparative metrics
- Percentile rank by season
- Era-adjusted z-scores
- League-relative payroll index
